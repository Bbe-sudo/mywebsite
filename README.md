# Digital Health Record System

## 📌 Project Description
A web-based system designed to manage patient records efficiently and securely. This project simulates how hospitals store and retrieve patient data.

---

## Key Features
- Role-Based Access Control (RBAC)
- Secure login authentication
- Data validation module
- Database integration for storing patient records

---

## 🛠 Tech Stack
- HTML
- CSS
- SQL
- PHP (Basic backend logic)

---

## 📊 Impact
- Improved record accuracy in simulated environment
- Demonstrated secure access handling for 100+ users

---

## 📂 Project Structure
- `create_record.php` – Handles record creation
- `add_patient.css` – Styling for patient form
- `database.sql` – Database schema
- Other supporting files
